# Perl

My collection of Perl files.

## How to install Perl.

Most GNU/Linux or \*BSD based operating systems have Perl already installed in them.

But: 

### Archlinux.

pacman -Syy base

or

pacman -Syy perl

### Debian/Ubuntu.

apt install -y perl

### CentOS/RHEL.

yum install -y perl

### Fedora.

dnf install -y perl

### Gentoo.

emerge -v perl

### openSUSE.

zypper install -y perl

### Solus.

eopkg it perl

## To install a local Perl interpreter in $HOME using Perlbrew🍺.

curl -L http://xrl.us/installperlnix | bash

## How to run these files.

Mark as executable.

`chmod +x filename.pl`

Run.

`perl filename.pl`
